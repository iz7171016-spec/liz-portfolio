# Liz 网站文字怎么改

以后只需要打开这个文件：

`content.js`

改文字时记住三件事：

1. 只改引号里面的文字。
2. 不要删除逗号、冒号、大括号 `{}`、中括号 `[]`。
3. 改完按 `Ctrl + S` 保存，然后刷新网页。

## 常见要改的位置

### 改网站名字

```js
brand: "Liz",
```

### 改首页三行英文

```js
hero: {
  lines: [
    "CREATIVE DESIGNER",
    "AI VISUAL CREATOR",
    "BUILD AESTHETIC DIGITAL WORKS"
  ]
}
```

### 改关于我

```js
about: {
  titleLine1: "大家好，",
  titleLine2: "我是",
  highlightName: "Liz",
  description: "这里改你的个人介绍。"
}
```

### 改作品标题和介绍

```js
{
  title: "Liquid Memory",
  category: "AI创意",
  image: "./images/work-1.jpg",
  description: "这里改作品介绍。"
}
```

### 改邮箱

```js
email: "hello@example.com",
```

如果网页刷新后没变化，先确认文件已经保存，再按 `Ctrl + F5` 强制刷新。
